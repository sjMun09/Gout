# Gout Care — 통풍 환자 전용 포털

통풍 환자를 위한 웹/앱 서비스

## 구조

- `gout-front/` — Next.js 15 (PWA, TypeScript, Tailwind)
- `gout-back/` — Spring Boot 3 (JPA, QueryDSL, JWT, PostgreSQL)

## 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Next.js 15, TypeScript, Tailwind CSS, shadcn/ui |
| Backend | Spring Boot 3, JPA, QueryDSL, Spring Security, JWT |
| Database | PostgreSQL 17, PostGIS, pgvector |
| 인프라 | AWS, Docker, GitHub Actions |
